# Facebook Clone

A fully responsive Facebook-inspired user interface built from scratch using **HTML5, CSS3, and Vanilla JavaScript**.
This project focuses on recreating the modern social media experience with a clean layout, interactive components, and a smooth user interface.

## 🚀 Overview

Facebook Clone is a front-end project designed to demonstrate strong fundamentals in web development, including:

* Semantic HTML structure
* Advanced CSS styling and responsive layouts
* JavaScript-based user interactions
* Component-like UI organization

## ✨ Key Features

### 🎨 User Interface

* Modern Facebook-inspired design
* Responsive layout for desktop and mobile screens
* Clean navigation bar and sidebar sections
* Dynamic content presentation

### 📝 Posts System

* Create and display posts
* Interactive like functionality
* Comment interface
* Post cards with user information

### 👤 Profile Experience

* User profile layout
* Profile information section
* User activity display

### ⚡ JavaScript Interactions

* Dynamic UI updates
* Event handling
* Interactive buttons and elements
